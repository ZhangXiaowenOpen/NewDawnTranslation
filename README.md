胸口发闷、呼吸停顿、眼眶发热、轻轻一松。

用9条死命令、4阶段流程、3级检查矩阵、  
把所有“让读者舒服”的冲动公开处决。

文学、疗愈手册、私人书写、社会观察、  
任何你写的时候在颤抖、读的时候会落泪的文字。

它不适用于技术文档、法律合同、新闻快讯——  
那些地方需要的是“看懂”，不是“被打到”。

如果你也曾因为一篇翻译

让我们一起，把人类最细微、最顽固的温度，
