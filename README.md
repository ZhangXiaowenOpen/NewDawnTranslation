**AI幻觉不是能力问题，是约束问题。**

行业路径：更多数据 → 更大模型 → 更多算力 → 减少幻觉

我们的路径：显式意图 → 行为约束 → 多版本验证 → 幻觉消失
```
P(hallucination) ∝ U(intent)
意图不确定性趋近于零，幻觉概率趋近于零。
```
